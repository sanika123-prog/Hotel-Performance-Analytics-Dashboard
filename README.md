# 🏨 Hotel Sales Analysis Dashboard

## 📌 Project Overview

An interactive Excel dashboard designed to analyze hotel sales performance across different states, booking channels, room types, and time periods. The dashboard provides actionable insights through dynamic visualizations and slicer-based filtering.

## 🎯 Objectives

* Analyze hotel revenue trends.
* Track booking performance.
* Identify top-performing states and branches.
* Evaluate booking channel effectiveness.
* Monitor room-type revenue contribution.

## 📊 Key Metrics

* Total Revenue: ₹14.91 Billion
* Average Revenue per Booking: ₹49,716
* Top Performing State: Karnataka

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Map Charts

## 📈 Dashboard Features

* Revenue KPI Cards
* Quarterly Revenue Trends
* Branch Performance Analysis
* Booking Channel Analysis
* Monthly Revenue Trends
* Room Type Revenue Distribution
* State-wise Revenue Map

## 💡 Key Insights

* Karnataka generated the highest revenue.
* Q4 recorded the highest revenue among all quarters.
* Booking channels showed varying revenue contributions.
* Revenue trends helped identify peak-performing months.
